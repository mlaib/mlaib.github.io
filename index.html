Environmental Data Analysis

In recent decades, understanding environmental data using machine learning and novel statistical techniques has gained great popularity. Environmental phenomena are in many cases complex, non-linear and multivariate. For instance, wind speed in Switzerland is characterized by extreme behaviour and is usually recorded with several sensors in the same time. Studying such phenomenon is still a challenging issue. Among the proposed methodologies here are:
• Network analysis to investigate the wind monitoring system.
• Spatial mapping of multifractal parameters obtained from each wind speed time series.
• Spatial modelling of extreme wind speed distribution using extreme learning machine.
• Unsupervised feature selection to reduce the existing redundancy in the input space.

Some R libraries

Since I am have a great interest in R programming, I propose some libraries that I developed in collaboration with my colleagues.
